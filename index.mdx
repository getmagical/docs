## Learning path

1. Start with [Navigate Magical](/basics/navigation).
2. Move to [Agents and Commands](/basics/agents-and-commands).
3. Build from [Manage and Create](/automations/manage-and-create) through [Node Guides](/automations/node-guides).
4. Validate with [Draft Runs and Tests](/automations/draft-runs-and-tests).
5. Ship with [Publish and Versions](/automations/publish-and-versions).
6. Operate from the run and settings guides.

## Coverage

The current site covers:

- platform navigation and feature-flagged surfaces;
- automations, agents, tools, commands, testing, and publishing;
- run review, live supervision, checkpoints, evals, and notifications;
- shared assets, import/export, and workspace credentials;
- practice exercises and the March 2026 changelog.
